# Pico Development

This repo contains various projects for the Raspberry Pi Pico Dev Boards and MCU's. It is intended to be used with the VS Code extension for Raspberry Pi Pico Development.

### Projects

*  pico2w-dev - A project to test the functionality of the Raspberry Pi Pico W with an 8x8 WS2812B LED matrix, a NEO-6M GPS module, and a WiFi web server to interact from PC's on the same network.

* macgetter - A simple project that spits out the MAC address of the Pico W to the serial console on boot.

* pico2LCD - Original dev code for st7735 LCD

* pico-radar - Radar using HLK-LD2410C radar module and displaying radar status on LCD and hosting webserver.

* pico-fancy-lcd - Fancy new code for st7735 LCD using framebufferes for no flicker changes and running at 60 fps. Also contains web server code for changing and viewing animations.

* kb2040-tests - quick tests with adafruit kb2040 dev board and on its on-board rgb LED.
